# Jingchao Zhou's Personal Website

**Development**:
- Ensure that the branch is on **develop**.
- Run `npm install` and `npm install -g gatsby-cli` to install packages.
- Run `gatsby develop -H 0.0.0.0 -p PORT_NUMBER` to start the development server.
- Remember to commit and push the changes.

**Deployment**:
- Type `gatsby build` to build the pages.
- Type `./deploy.js` to deploy the pages on **github-pages**.
